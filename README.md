# ruby-ll

ruby-ll is an LL(1) parser generator for Ruby, it uses a native extension (C
for MRI/Rubinius and Java for JRuby) to ensure good parser performance.

## Requirements

* C99 (for MRI/Rubinius)
* Ruby 2 or newer
